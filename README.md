# Module I — Fundamental Python for Modern AI

Le code de la série **Python de zéro à production** sur la chaîne
[NounLab](https://youtube.com/@NounLab).

**Le code et l'IA comme on les écrit en production.**

Chaque dossier correspond à une vidéo : le code, les slides, et un README
qui explique ce que la vidéo couvre.

## Prérequis

- Python 3.12+
- [uv](https://docs.astral.sh/uv/) pour la gestion des dépendances
- VS Code (optionnel mais recommandé)

Pour démarrer un projet propre, utilisez le
[starter template](https://github.com/nounlabio/python-starter-template).
