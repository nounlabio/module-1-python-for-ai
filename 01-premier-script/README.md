# 01 — Votre premier script Python utile

Lire un relevé bancaire CSV et calculer les dépenses par catégorie,
en Python pur, sans dépendance externe.

## Notions couvertes

Variables · Boucles · Conditions · Dictionnaires · Conversion de types · f-strings

## Fichiers

- `analyse_depenses.py` — le script final
- `depenses_janvier.csv` — les données (247 transactions, anonymisées)
- `slides.md` — les slides Slidev

## Lancer

```bash
python analyse_depenses.py
```
